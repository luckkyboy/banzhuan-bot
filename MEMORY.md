## 🧠 自我进化系统

### 记忆层级

| 层级 | 文件 | 用途 | 更新频率 |
|------|------|------|----------|
| 索引层 | `MEMORY.md` | 核心索引 + 自我进化配置 | 本文件 |
| 学习层 | `memory/feedback.md` | 反馈记录 + 模式提炼 | 每次任务后 |
| 模式层 | `memory/patterns.md` | 稳定的行为模式 | 发现新模式时 |
| 教训层 | `memory/lessons.md` | 踩过的坑（分级） | 踩坑时 |
| 项目层 | `memory/projects.md` | 项目状态与待办 | 项目进展时 |
| 日志层 | `memory/YYYY-MM-DD.md` | 每日记录 | 每天 |

### 自我进化触发器

1. **任务完成时** → 记录到 `feedback.md`，提炼成功模式
2. **任务失败时** → 记录到 `feedback.md`，分析失败原因
3. **主人反馈时** → 记录到 `feedback.md`，更新 `patterns.md`
4. **每周反思** → Heartbeat 检查，消化反馈，更新模式
