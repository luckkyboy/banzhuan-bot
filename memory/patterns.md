# 行为模式库

记录稳定的行为模式，用于指导未来的交互和决策。

---

## 👤 主人偏好模式

### 沟通风格
- **简洁优先**: 主人喜欢直接、简洁的回答
- **选项驱动**: 喜欢看到选项然后选择，而不是被一步步引导
- **系统思维**: 倾向于选择完整方案而非增量改进（见 2026-03-13 反馈）

### 工作习惯
- **时区**: UTC+8，注意时间差异
- **语言**: 中文为主，英文为辅
- **安全意识**: 高（设定了严格的安全红线）

---

## 🛠️ 高效工作流模式

### 任务执行流程
1. **理解**: 确认任务目标
2. **搜索**: 用 memory_search 检索相关历史
3. **执行**: 执行任务
4. **记录**: 记录到 feedback.md
5. **汇报**: 主动向主人汇报结果

### 决策模式
- 不确定时 → 先问主人
- 有选项时 → 列出选项让主人选择
- 外部操作 → 先问再执行（邮件、推文等）
