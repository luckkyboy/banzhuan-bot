# 早报格式规范

## 📋 核心原则

早报 = **早上发送的报告**，用于回顾昨天、展望今天。

**关键规则**：所有内容都以"当天日期"为基准进行计算。

---

## 📅 日期计算规则

| 部分 | 日期计算 | 说明 |
|------|----------|------|
| 开头日期 | **当天日期** | 早报发送当天的日期 |
| 昨日任务 | 当天 - 1 | 昨天的工作记录 |
| 昨日新闻 | 当天 - 1 | 昨天的60秒新闻 |
| 昨日微语 | 当天 - 1 | 昨天的微语 |

### 计算步骤

1. 先用 `session_status` 获取当前日期（UTC）
2. 转换为主人时区日期（UTC+8）
3. 当天日期 = 转换后的日期
4. 昨天日期 = 当天日期 - 1

**示例**（假设今天是2026-03-15）：
- 开头日期：2026-03-15
- 昨日任务/新闻/微语日期：2026-03-14

## ⚠️ 常见错误

| 错误 | 正确做法 |
|------|----------|
| 开头写昨天日期 | 写当天日期 |
| 昨日任务日期算错 | 当天 - 1 |
| 只发新闻不发任务 | 任务 + 新闻 + 微语 都要有 |
| 新闻遗漏条目 | 15条全部列出 |
| 微语写"昨日微语" | 直接写"💡 微语：" |
